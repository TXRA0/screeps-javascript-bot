# Screeps Javascript bot

## Javascript bot

Basic AI for the game [**Screeps: World**](https://screeps.com/)  
Designed to work in any room, though currently only tested by myself.  
My profile is here: [**TXR**](https://screeps.com/a/#!/profile/_TXR)
